# Listas Genéricas e Ordenadas em C++

Projeto desenvolvido em C++ para implementação de listas genéricas e listas ordenadas utilizando templates.

## Objetivo

Implementar estruturas de dados reutilizáveis capazes de armazenar diferentes tipos de informações.

## Conceitos Aplicados

- Templates
- Programação Orientada a Objetos
- Vetores (std::vector)
- Sobrecarga de Operadores
- Busca Sequencial
- Inserção e Remoção de Elementos
- Estruturas de Dados

## Estruturas Implementadas

### Lista Genérica

Permite:

- Adicionar elementos
- Inserir elementos em posições específicas
- Remover elementos
- Buscar elementos
- Acessar elementos por índice

### Lista Ordenada

Mantém os elementos automaticamente ordenados durante a inserção.

## Exemplos Utilizados

### Lista de Inteiros

```cpp
Lista<int>
```

### Lista de Tarefas

```cpp
ListaOrdenada<tarefa>
```

### Lista de Compras

```cpp
Lista<compras>
```

## Tecnologias

- C++
- STL (Standard Template Library)
